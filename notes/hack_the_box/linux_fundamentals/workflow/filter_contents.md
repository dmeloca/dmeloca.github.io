# Filter Contents
## More
This manage the output as a phone directory (pager).
## Less
The command is similar to `more` but with more features.
## Head
Useful to print the first line of a file (by default it print the first 10 lines)
## Tail
Same as head but it prints the last lines of a file.
## Sort
Helps us to sort the output (i.e. sorting result alphabetically)
## Grep 
This command let us search for specific things in the input given.
## Cut
Helps managing input with delimiters. `cut -d":" -f1` which prints the string before the first `:`.
## Tr
Makes a find and replace. `tr ":" " "`
## Column
Helps organizing the input, for example adding tabs to make a table with the input.
## Awk
This helps us displaying specific parts of the input. `awk {print $1, $NF}`, prints the first and the last element.
## Sed
Another find and replace but with regex.
## Wc
Helps counting the lines that an input has.
