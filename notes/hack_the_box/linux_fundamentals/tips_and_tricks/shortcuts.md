# Shortcuts
| Categoría | Atajo | Función |
|----------|-------|---------|
| Auto-complete | TAB | Inicia el autocompletado según el contexto (comandos, directorios u opciones). |
| Movimiento del cursor | CTRL + A | Mueve el cursor al inicio de la línea. |
| Movimiento del cursor | CTRL + E | Mueve el cursor al final de la línea. |
| Movimiento del cursor | CTRL + ← / → | Salta al inicio de la palabra actual o anterior. |
| Movimiento del cursor | ALT + B / F | Salta una palabra hacia atrás o hacia adelante. |
| Borrar línea | CTRL + U | Borra desde el cursor hasta el inicio de la línea. |
| Borrar línea | CTRL + K | Borra desde el cursor hasta el final de la línea. |
| Borrar palabra | CTRL + W | Borra la palabra anterior al cursor. |
| Pegar texto | CTRL + Y | Pega el último texto borrado. |
| Finalizar proceso | CTRL + C | Termina el proceso actual enviando la señal SIGINT. |
| Fin de archivo | CTRL + D | Cierra la entrada estándar (EOF). |
| Limpiar terminal | CTRL + L | Limpia la pantalla del terminal. |
| Procesos | CTRL + Z | Suspende el proceso actual (SIGTSTP). |
| Historial | CTRL + R | Busca comandos previos en el historial. |
| Historial | ↑ / ↓ | Navega entre comandos anteriores y siguientes. |
| Aplicaciones | ALT + TAB | Cambia entre aplicaciones abiertas. |
| Zoom | CTRL + + | Aumenta el zoom del terminal. |
| Zoom | CTRL + - | Reduce el zoom del terminal. |
