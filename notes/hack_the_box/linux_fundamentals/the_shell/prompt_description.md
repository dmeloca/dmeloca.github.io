# Prompt Description
- `<username>@<hostname><current working directory>$`: Basic Bash prompt
- `~` refers to the user default folder
- `$` stands for user
- `#` stands for root
## PS1 variable
In Linux controls how the command prompt will be displayed. This allows to personalize the interface.

| Special Character | Description                                |
|-------------------|--------------------------------------------|
| `\d`              | Date (Mon Feb 6)                           |
| `\D{%Y-%m-%d}`    | Date (YYYY-MM-DD)                          |
| `\H`              | Full hostname                              |
| `\j`              | Number of jobs managed by the shell        |
| `\n`              | Newline                                    |
| `\r`              | Carriage return                            |
| `\s`              | Name of the shell                          |
| `\t`              | Current time 24-hour (HH:MM:SS)            |
| `\T`              | Current time 12-hour (HH:MM:SS)            |
| `\@`              | Current time                               |
| `\u`              | Current username                           |
| `\w`              | Full path of the current working directory |
