## HTTP Request
![Request Parts](../../images/request_parts.png)
- Method: type of connection to perform
- Path: resource to be accessed
- Version: Denote the http version
> HTTP version 1.X sends requests as clear-text, and uses a new-line character to separate different fields and different requests. HTTP version 2.X, on the other hand, sends requests as binary data in a dictionary form.

## HTTP Response
![Response parts](../../images/response_parts.png)
- Respond codes: determine the request's status
- Respond Body: The information retrieved from the server
